# Task Manager (Python CLI Application)

A simple and lightweight Task Manager built using Python.  
This project helps users add, edit, delete, and manage everyday tasks using a clean command-line interface.  
All data is saved locally using JSON, so your tasks remain safe even after restarting the program.
